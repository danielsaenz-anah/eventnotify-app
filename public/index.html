Arquitectura

Una estructura diseñada para separar responsabilidades

El sistema desacopla la publicación de eventos, la creación de notificaciones y el envío por canal, lo que permite una solución más modular y fácil de extender.

Facade

Centraliza la interacción del sistema desde un solo punto.

Observer

Permite notificar automáticamente a todos los suscriptores registrados.

Factory + Strategy

Define el canal correcto y encapsula el comportamiento de envío.
